# Changelog

Все заметные изменения в этом проекте документируются в этом файле.

## [0.1.0] - 2026-01-16

### Добавлено (Added)
- Добавлены функции для подготовки данных к отправке/получении по сети: `EncryptNetworkData` и `DecryptNetworkData`

- Поддержка `netstandard2.1`, `net8.0`, `net9.0`

- **Система логирования** с возможностью подписки на события через свойство `Log`

### Изменено (Changed)
- Теперь вы сами можете указать свою стартовую папку при инициализации PlayerData

- Подправлены функции хеширования
